Description of cueMLv2.1

CueMLv2.1 is an ontology for recipes within cookbooks and based on TEI: Text Encoding Initiative and schema.org/Recipe. TEI is... Schema.org/Recipe is... This website should give an overview of it and provide examples of how to use it.

Example 1

Element Description Attribute Allowed child elements
cookTime The time the recipe has to cook/bake/... dur-iso text
head name of the recipe - text
link A reference to another source like a previous recipe target text
note Additional note to the recipe, as you can use white french wine and rum instead of madeira wine - cookTime, link, prepTime, recipeIngredient, recipeYield, text, totalTime
p (Part of) Textual description of the recipe - cookTime, link, prepTime, recipeIngredient, recipeYield, text, totalTime
prepTime The time it takes to prepare the recipe dur-iso text
recipe A recipe for a meal rcp-id, type, cookMethod head, p, note
recipeIngredient A single ingredient used in the recipe, e.g. sugar, flour or garlic isOptional, isAlt, dontUse, reference unit text
recipeYield Number of people served by the recipe quantity text
span Defines the start and end of a reference. id, from and to have to be a valid id in the document id, from, to text
spanGrp Container for all span-elements - span
text Plain text - -
totalTime The total time it takes to prepare and cook the recipe dur-iso text

Attribut Value Description
cookMethod steam, bake The main method of cooking, such as steamed fished
dontUse True States explicitly to not use this ingredient
dur-iso Duration in ISO_8601 Period of time
isAlt identifier Within a recipe all ingredients with the same isAlt-identifier are alternatives to each other
isOptional True States the ingredient is optional in the recipe
quantity double[-double][ pP] Specifies the quantity by a number or a range and optional states that it is meant as quantity per person (pP)
rcp-id String Unique id across the document
reference identifier The identifier of the reference contains more information for the ingredient. For example when the ingriedient is dumbling, a reference could target a recipe for dumblings. All ingredients of the reference are meant to be included as sub ingredients to the ingredient. Note that a dontUse-attribute is meant to hava a higher priority than the included sub ingredients
target id The id targets a span-element, which should define a reference
type String Specifies a meal-category such as soup or fish
unit Maß, Pfund, ... Unit of specified quantity
